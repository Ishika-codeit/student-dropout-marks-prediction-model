# student-dropout-marks-prediction-model
Developed a machine learning–based system to predict student dropout risk and academic performance using classification models. Designed a recommendation engine to suggest optimal study time, lecture completion, and assignment targets based on predicted marks range

# Project Overview
The Student Performance Analysis and Recommendation System is a machine learning–based project designed to analyze student learning behavior and predict academic outcomes. The system identifies potential dropout risks using classification models and predicts performance levels (marks range) based on engagement and course-related factors.
In addition, a recommendation module suggests optimal study time, lecture completion percentage, and assignment targets required to achieve a desired performance level. The project integrates data preprocessing, feature engineering, model training, and evaluation to build an end-to-end predictive analytics pipeline.
